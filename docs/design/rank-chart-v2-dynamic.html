Ranking v2 — altura dinâmica · rótulos = hover · mobile

As três mudanças pedidas, prototipadas: (1) o gráfico só cresce com quantos jogadores estão selecionados (poucos → baixinho); (2) o painel de nomes é o próprio hover — passar o mouse numa linha torna a lista dinâmica (posições/pontos daquele jogo) e acende a linha; (3) repensar o mobile.

Dados fictícios (40 jogadores, jogo a jogo). Você (amarelo) está em ~20º. Mude a seleção pra ver a altura mudar; passe o mouse no gráfico (desktop) ou arraste o controle (mobile). Escolha 1 desktop + 1 mobile.

Desktop A · Painel vivo Desktop B · Rótulos na linha Mobile A · Corrida Mobile B · Sparklines Mobile C · Bump + ficha protótipo — dados fictícios
DESKTOP A

Painel vivo (recomendado)

O gráfico fica enxuto (altura ∝ nº de selecionados) e os nomes saem de cima das linhas pra um painel à direita. Em repouso ele mostra a classificação final; ao passar o mouse, o painel vira o hover: reordena pro jogo sob o cursor, mostra os pontos e destaca a linha apontada. Sem tooltip flutuante.

DESKTOP B

Rótulos na linha que viram dinâmicos

Mantém o nome cravado na ponta de cada linha (leitura direta), mas sem tooltip: ao passar o mouse, os próprios rótulos viram dinâmicos — número e pontos mudam pro jogo sob o cursor e a linha apontada acende. Também com altura dinâmica.

Mobile — 3 caminhos

Linha-espaguete não cabe num celular. As três trocam o “seguir 40 linhas finas” por algo tátil. Todas respeitam: poucos selecionados → compacto; e a leitura é por toque/arraste, não por mira fina.

Mobile A · Corrida — arrasta o tempo, a lista ultrapassa
Mobile B · Sparklines — uma linha mini por jogador
Mobile C · Bump + ficha — gráfico pequeno, ficha embaixo

Desktop A · Painel vivo

  • Hover nunca cobre as linhas (painel fixo)
  • Rótulos = hover, de verdade
  • Altura dinâmica fácil (nomes saíram do plot)
  • Vínculo linha↔nome é por cor/realce, não espacial

Desktop B · Rótulos na linha

  • Leitura direta (nome na ponta)
  • Sem painel extra — mais largura pro gráfico
  • Rótulos se mexem ao reordenar no hover
  • Seleção larga reaperta as pontas

Mobile

  • A Corrida: tátil, conta a virada, escala bem
  • B Sparklines: tudo de relance, sem tangle
  • C Bump+ficha: mantém o gráfico, ficha no toque
